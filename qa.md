# Flexion EPA Prototype QA Documentation

## Supported browsers

The design is relying on the use of Flexbox and subsequently only going to be supported in the following browsers:

- Internet Explorer 11
- Microsoft Edge (12+)
- Firefox 41+
- Chrome 43+
- Safari 9+
- iOS 9.2+
- Android 4.4.4+
- Chrome for Android 47+
